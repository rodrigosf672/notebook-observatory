A daily census of the world's computational notebooks

Notebook Observatory samples public Jupyter notebooks from GitHub every day, parses their structure, detects the libraries they use, and publishes an ever-growing longitudinal dataset. Think Our World in Data, for notebooks.

The trend charts below use a historical backfill spanning 2014–2026: for each year, a stratified sample of notebooks whose repository was created that year, measured as they exist today. This is a creation-year cohort view — see the methodology for its survivorship caveat.

Ask the observatory

A small open-weight language model (Qwen2.5-0.5B-Instruct), grounded in the latest observatory data, answers questions about the notebook ecosystem — which libraries dominate, how adoption shifts by creation-year cohort, and what the dataset does and doesn't cover. It reasons only from the figures released here, so it will say when something is outside the data.

The agent runs on a free Hugging Face Space (CPU) and may take a few seconds to reply, or briefly sleep when idle. It is a research demo, not an authoritative source — verify against the datasets. Open the agent in a new tab ↗
